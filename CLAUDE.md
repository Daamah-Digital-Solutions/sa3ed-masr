## Project Overview

Sa3ed Masr ("Egypt Helps") is a mentorship platform connecting mentors with entrepreneurs/project seekers. Built with React 18 + TypeScript, Vite, shadcn-ui, Tailwind CSS, and Supabase backend.

## Commands

```bash
npm run dev          # Start dev server on port 8080
npm run build        # Production build
npm run lint         # Run ESLint
npm run test         # Run tests once
npm run test:watch   # Run tests in watch mode
```

## Architecture

### Tech Stack
- **Frontend**: React 18 + TypeScript + Vite
- **UI**: shadcn-ui (Radix UI + Tailwind CSS)
- **State**: TanStack React Query for server state
- **Forms**: React Hook Form + Zod validation
- **Backend/Auth**: Supabase (PostgreSQL + Auth with Google OAuth)
- **Platform**: Lovable-generated project

### Key Directories
- `src/pages/` - Route pages (Index, Auth, Profile, MentorRegister, SeekerRegister, BrowseMentors, BrowseProjects)
- `src/components/ui/` - shadcn-ui components (67 files)
- `src/components/layout/` - Layout, Header, Footer, UserMenu
- `src/components/cards/` - MentorCard, SeekerCard
- `src/hooks/` - useAuth, useTheme, use-mobile, use-toast
- `src/integrations/supabase/` - Supabase client and auto-generated types
- `src/lib/constants.ts` - Domain constants (expertise areas, help types, project stages)

### Routes
- `/` - Home page
- `/auth` - Google OAuth authentication
- `/profile` - User profile (mentor/seeker)
- `/mentor-register` - Register/edit as mentor
- `/seeker-register` - Register/edit as project seeker
- `/mentors` - Browse mentors with filtering
- `/projects` - Browse projects with filtering

### Database Tables
- `mentors` - Mentor profiles
- `seekers` - Project seeker profiles

## Development Notes

- **Arabic RTL**: UI uses "IBM Plex Sans Arabic" font with right-to-left layout
- **Dark mode**: Supported via next-themes
- **Auto-generated files**: Do not edit `src/integrations/supabase/types.ts` directly (auto-generated by Lovable/Supabase)
- **TypeScript config**: Permissive settings (noImplicitAny: false, strictNullChecks: false)
- **Environment**: Supabase credentials in `.env` with `VITE_` prefix

## Domain Constants

Located in `src/lib/constants.ts`:
- Expertise areas: Finance, Marketing, Programming, AI, Operations, Sales, Product, Legal, Fintech, Growth
- Help types: Mentorship, Training, Funding, Partnerships, Hiring, Consulting
- Project stages: Idea, MVP, Early, Growth, Scale
